# 📊 Retail Sales Performance Dashboard

## 📌 Project Overview

This project presents an end-to-end Retail Sales Performance Analysis using **SQL**, **Power BI**, **DAX**, and **Excel**. The objective is to transform raw retail transaction data into meaningful business insights through data cleaning, SQL analysis, and interactive dashboards.

The dashboard enables business users to monitor sales performance, analyze profitability, identify top-performing customers and products, and make data-driven decisions.

---

## 🎯 Business Objectives

- Analyze overall sales and profit performance
- Identify top customers and products
- Monitor sales trends over time
- Compare regional and category performance
- Measure profit margins
- Support business decision-making using interactive dashboards

## 🧹 Data Preparation

Before creating the dashboard, the dataset was cleaned using SQL.

Data cleaning tasks included:

- Removing leading/trailing spaces
- Standardizing text values
- Handling NULL values
- Correcting invalid numeric values
- Removing duplicate records
- Converting data types
- Standardizing date formats

## 📈 SQL Analysis

Business questions answered using SQL include:

- Top 10 Customers by Sales
- Top 10 Products by Profit
- Monthly Sales Trend
- Revenue by Category
- Sales Contribution by Region
- Customer Ranking using Window Functions

- Duplicate Detection and Data Cleaning

## 💡 Key Business Insights

- Consumer segment generated the highest sales.
- Technology category contributed the highest profit.
- Regional performance varies significantly across markets.
- A small number of customers contribute a large portion of revenue.
- Sales show consistent growth across the analyzed years.
